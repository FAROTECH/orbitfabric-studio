# OrbitFabric Studio v0.11.1 Release Notes

OrbitFabric Studio v0.11.1 is a runtime UI triage hotfix for the v0.11.0 Domain Surfaces & Entity Detail System milestone.

This release does not introduce v0.12.0 functionality.

## Fixed

- Reset the main content scroll position when navigating between Studio surfaces.
- Clarified source file actions across domain surfaces by replacing ambiguous `Preview <file>` wording with `Inspect source file: <file>`.
- Added an explicit reserved Autonomy surface instead of falling back to the generic legacy Model Inventory surface.
- Corrected the Mission Cockpit Scenario / Recent runs empty-state copy when the scenario run index report exists but contains no indexed runs.

## Preserved boundaries

- No Mission Data Flow Workbench.
- No React Flow or graph UI.
- No authoring.
- No plugin behavior.
- No command uplink.
- No live telemetry.
- No Autonomy implementation.
- No semantic mutation of generated artifacts.
