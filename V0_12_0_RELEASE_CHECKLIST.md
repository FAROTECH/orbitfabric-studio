# OrbitFabric Studio v0.12.0 Release Checklist

Release: `v0.12.0 - Mission Data Flow Workbench Foundation`

## Scope confirmation

- [ ] Mission Data Flow Workbench model foundation is present.
- [ ] Mission Data Flow Workbench Reference B layout is present.
- [ ] Dedicated Mission Data Flow Workbench route is wired into the Studio shell.
- [ ] Data Flow Workbench sidebar entry is available.
- [ ] Inspect Data Flow command bar action is available.
- [ ] Workbench renders Core-derived sources, lanes, records, counts and boundaries.
- [ ] Workbench canvas renders nodes and edges only from Core-reported relationship manifest records.
- [ ] Workbench scenario evidence renders only Core simulation report `data_flow_evidence` records.
- [ ] Workbench coverage renders only Core coverage summary records.
- [ ] Relationship manifest persistence works across Core report switches.
- [ ] Relationship manifest state resets on workspace change.
